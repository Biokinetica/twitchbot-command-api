Twitch Chat Command API

Hey, how's it going? This is an api for twitch chat commands. Use the $readapi() command for the endpoints in Streamlabs chatbot.

The endpoints are currently:

  • /weather - https://twitch-command-api.herokuapp.com/weather/?city=$msg
  • /movie - https://twitch-command-api.herokuapp.com/movie?title=$msg
  • /time - https://twitch-command-api.herokuapp.com/time/?city=$target&region=$target2

The movie endpoint can take a movie title and year, or just the title. The weather endpoint will take the city and region, or try it's best with city and country. Everything is case-insensitive. Just use spaces between words and type naturally!

The weather endpoint can take the option -fc for forecast. It will get the high and low for the specified day. Example: !weather -fc Tue boden norrbotten

The movie endpoint can also take -cast or -plot as options. For example: !movie -cast Marshall 2017
